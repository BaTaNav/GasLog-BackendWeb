Users Endpoints

GET /users

Haalt een lijst van alle gebruikers op. Kan gesorteerd worden met query parameters.

Query parameters:

  • sort - Sorteer veld: id, username, created_at (default: id)
  • order - Sorteer richting: asc, desc (default: asc)

GET voorbeelden:

  • /users?sort=username&order=asc - Sorteer alfabetisch op username
  • /users?sort=created_at&order=desc - Nieuwste gebruikers eerst

GET /users/:id

Haalt de details van een specifieke gebruiker op door ID.

URL parameter: id (integer)

POST /users

Voegt een nieuwe gebruiker toe. Het wachtwoord wordt gehashed opgeslagen.

Body JSON:

{
  "username": "broer",
  "password": "1234"
}

PUT /users/:id

Update een bestaande gebruiker. Kan username of password aanpassen.

Body JSON voorbeeld:

{
  "username": "newname",
  "password": "newpass"
}

DELETE /users/:id

Verwijdert een gebruiker op basis van ID.

Fuel Entries Endpoints

GET /fuel

Haalt een lijst van alle tankbeurten op. Kan worden gefilterd, gesorteerd en gepagineerd met query parameters.

Query parameters:

  • limit - Aantal resultaten (default: 10)
  • offset - Start positie (default: 0)
  • user_id - Filter op gebruiker (optioneel)
  • car_id - Filter op auto (optioneel)
  • date - Filter op datum (optioneel)
  • sort - Sorteer veld: date, liters, price_per_liter, odometer, total_amount, created_at (default: date)
  • order - Sorteer richting: asc, desc (default: desc)

Response voorbeeld:

[
  {
    "id": 1,
    "car_id": 1,
    "user_id": 3,
    "date": "2025-01-12",
    "liters": 42.5,
    "price_per_liter": 1.79,
    "odometer": 128400,
    "total_amount": 76.075,
    "created_at": "2025-12-25T10:23:45.390Z"
  }
]

GET /fuel/:id

Haalt een specifieke fuel entry op door ID.

POST /fuel

Voegt een nieuwe fuel entry toe voor een specifieke auto en gebruiker.

Body JSON voorbeeld:

{
  "car_id": 1,
  "user_id": 3,
  "date": "2025-01-12",
  "liters": 42.5,
  "price_per_liter": 1.79,
  "odometer": 128400
}

PUT /fuel/:id

Update een bestaande fuel entry. Alleen opgegeven velden worden aangepast.

Body JSON voorbeeld:

{
  "liters": 45,
  "price_per_liter": 1.80,
  "odometer": 128500,
  "date": "2025-01-13"
}

DELETE /fuel/:id

Verwijdert een fuel entry op basis van ID.

Filter & Pagination

Je kunt fuel entries filteren op bijvoorbeeld user_id en pagineren met limit en offset:

GET voorbeelden:

  • /fuel?user_id=3&limit=10&offset=0 - Filter op user, max 10 resultaten
  • /fuel?sort=liters&order=desc - Sorteer op liters, hoogste eerst
  • /fuel?car_id=1&sort=date&order=asc - Filter op auto, sorteer op datum oplopend
  • /fuel?user_id=3&sort=total_amount&order=desc&limit=5 - Top 5 duurste tankbeurten van user 3
